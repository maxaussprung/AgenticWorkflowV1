# Test coverage

Per-requirement test coverage, generated by the test-author agents and rendered from
`reports/test-coverage/<layer>/<REQ-ID>.json`. Each requirement page links here to its detailed
section: mechanical coverage, the acceptance-criteria breakdown, what changed, and what is not
yet fulfilled.

Replace the per-layer paths and file format to match your project's test reporting structure.

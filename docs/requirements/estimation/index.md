# estimation/

This folder holds **effort estimation data** for features and epics — sizing records, story point tallies, complexity assessments.

## Expected Files

- `{EPIC-ID}-estimation.md` or `{FEAT-ID}-estimation.md` — per-item sizing records
- `summary.md` — aggregate totals across all features

## Format

Each estimation file should include: feature/epic ID, estimated effort (days/points), confidence level, and assumptions.

# page-types/

This folder defines **UI page type patterns** — reusable screen archetypes that appear across the application (e.g. list page, detail page, form page, dashboard).

## Expected Files

- `{PAGE-TYPE-ID}.md` — one file per page type pattern

## Format

Each file should describe: the page type's purpose, common UI components, navigation behaviour, and which requirements/features use this pattern.

# Tests

This folder holds automated tests for the target application.

## Structure

- `{ClientName}.{ProjectName}.UnitTests/` — xUnit unit tests
- `{ClientName}.{ProjectName}.IntegrationTests/` — Integration tests against real infrastructure
- `{ClientName}.{ProjectName}.E2ETests/` — End-to-end tests (Playwright or similar)
- `{ClientName}.{ProjectName}.TestUtilities/` — Shared test helpers and fixtures

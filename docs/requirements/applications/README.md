# applications/

This folder lists all **external systems and applications** that the migrated system integrates with or depends on.

## Expected Files

- `{APP-NAME}.md` — one file per application

## Format

Each file should describe: the application's purpose, integration type (API, database, event bus), and which requirements depend on it.

## Naming

Use the application's canonical name or abbreviation as the filename (e.g. `SAP.md`, `Kafka.md`, `KeyCloak.md`).

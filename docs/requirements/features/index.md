# Features

Coherent slices of capability under a product. Each feature groups a set of related requirements and links to the epics and products it contributes to.

## Example

- [FEAT-EXAMPLE-001 — Example Feature: Advanced Search](FEAT-EXAMPLE-001.md)

## ID Pattern

Features follow the pattern `FEAT-<NNN>` where `<NNN>` is a zero-padded sequence number (e.g. `FEAT-001`).

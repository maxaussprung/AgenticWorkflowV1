# Source traceability

A consolidated, all-products view: every product, the features it consists of, and the
sources each feature was derived from. The **highest** source tier per feature is shown as a
tag — **gold** (backlog / user story) > **silver** (specification document) > **bronze** (legacy code / analysis).
The *Sources* column shows every source the feature draws on, in that priority order. Where
sources conflicted, the resolution is recorded on the feature page itself. Legacy database/SQL links, where
present, are shown as supporting evidence only; they do not raise the source tier.
